# Personal Introduction Program

## Project Overview
This program asks the user for their name, age, and hobby and prints a friendly introduction message.

## How to Run
1. Open Command Prompt
2. Navigate to project folder:

cd OneDrive\Desktop\Personalintroduction
3. Run the program:

python week1code.py
